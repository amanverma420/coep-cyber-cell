## 📌 Overview

A modern, responsive website for the College of Engineering Pune (COEP) Cyber Cell, built with React, TypeScript, and Tailwind CSS.  
It features event showcases, team profiles, contact forms, and more — designed with speed, accessibility, and scalability in mind.

🔗 **Live Demo** → https://your-demo-link.vercel.app (Replace with your deployed URL)

## 🚀 Features

- 🎨 **Modern UI/UX** with shadcn/ui + Tailwind CSS
- 📱 **Responsive design** for mobile, tablet, and desktop
- ⚡ **Fast performance** powered by Vite
- 🔒 **Type safety** with full TypeScript support
- 🧩 **Reusable components** for scalability
- 📅 **Events showcase**, 👥 **Team profiles**, and 📬 **Contact form**
- 🌐 **SEO-friendly** and accessible design

## 🖥️ How to Run Locally

### Clone the repository
```bash
git clone https://github.com/amanverma420/coep-cybercell.git
cd coep-cybercell
```

### Install dependencies
```bash
npm install
```
or
```bash
bun install
```

### Start the development server
```bash
npm run dev
```
or
```bash
bun dev
```

### Open in browser
```
http://localhost:8080
```
